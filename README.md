# StyleCart Django Ecommerce

A professional Django ecommerce starter project with:
- Login / Signup / Logout
- Navbar and footer
- Category pages
- Product list and product detail page
- Slider on homepage
- Cart and checkout
- Order place and order history
- Media image upload support
- Admin panel to add your own categories and product images

## Setup

```bash
python -m venv venv
# Windows
venv\Scriptsctivate
pip install -r requirements.txt
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
```

Open:
- http://127.0.0.1:8000/
- http://127.0.0.1:8000/admin/

## Add your own images
1. Login to admin panel.
2. Create categories.
3. Create products.
4. Upload your own image in the Product image field.
